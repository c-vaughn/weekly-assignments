
# ASSIGNMENT 2: Login/Oauth

### Setup
clone repo

**Backend:**

```
cd weekly-assignments
npm install

node server.js 
```
(starts on [http:localhost:3000](http:localhost:3000))

**Frontend:**

```
cd client
npm install

npm s
```
(starts on [http:localhost:3001](http:localhost:3001))

Must start here as CORS is configured for localhost:3001
